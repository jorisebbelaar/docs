# FastAPI

<Info>
The **FastAPI framework** is utilized for creating endpoints and handling authentication and other HTTP communication. Its tight integration with **Pydantic** makes data validation and serialization seamless.
</Info>

<Card title="Custom Endpoints" icon="plug">
Within **Launchpad**, you can define custom endpoints to handle specific event types. This flexibility allows integration of custom behavior and makes it easy to extend or adapt functionality.
</Card>

## Key Features

  <Card title="Async Support" icon="bolt">

  <Card title="OpenAPI Integration" icon="book-open">

  <Card title="Pydantic Integration" icon="link">

  </Card>
</CardGroup>

FastAPI's seamless integration with Pydantic makes it an ideal choice for GenAI applications requiring robust data validation and API documentation.
